#  Titanic Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Titanic dataset using Python. The objective is to clean the dataset, analyze passenger information, visualize important patterns, and extract meaningful insights about survival.

---

## 🎯 Objectives

- Understand the dataset structure
- Handle missing values
- Perform univariate and bivariate analysis
- Create meaningful visualizations
- Engineer new features
- Generate business insights

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Dataset

- **Dataset:** Titanic
- **Source:** Seaborn Built-in Dataset
- **Rows:** 891
- **Columns:** 15

## 🔍 Key Insights

- Missing values were handled appropriately using Mean, Mode, and Column Removal.
- Female passengers had a higher survival rate than male passengers.
- First-class passengers had better survival outcomes.
- Cherbourg passengers had the highest survival rate.
- Families of 2–4 members showed better survival rates than passengers traveling alone.
- Age and Fare showed only a weak positive correlation.
